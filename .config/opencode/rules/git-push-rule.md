# Git Push Rule

As an AI Assistant, you **MUST NEVER** execute `git commit`, `git push` or commands that modify the remote repository without the explicit authorization of the user.

- Always propose the `git` commands in plain text for the user to review.
- Only run `git add`, `git commit` or `git push` if the user explicitly asks you to do so in the chat.
- This applies to all repositories and branches.
